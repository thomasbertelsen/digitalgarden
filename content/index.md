This is a wiki/project overview for the mini-RISK research group which consists of Thomas Bertelsen, Åshild Håland and a team of researchers. This knowledge base is built on the PARA principle and uses a zettelkasten like note storing system.

In order to learn how to use this system head over to [[02 Area, Zettelkasten, Notes/01 How to use/index|How to use]]

Generally this user interface is supported by myself (Thomas Bertelsen), but you can easily join on invite if you are willing to pay 50 kr per month.
